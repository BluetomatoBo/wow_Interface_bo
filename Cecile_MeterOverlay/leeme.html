Cecile Meter Overlay : Léeme
Este complemento es una versión mejorada de Elvui_MeterOverlay, diseñado originalmente para mostrar la información de cualquier medidor de combate en un texto de datos.

Muestra un texto de datos configurable con la información de tu medidor de combate, como por ejemplo tu dps o hps, esta información se puede personalizar mediante un sistema de etiquetas.

Al pasar el ratón sobre el texto una ventana flotante aparecerá con la información de tu grupo o banda.

Gracias a esto no necesitarás tener abierta la ventana de tu medidor de combate, simplemente pasa el raton por encima para ver la información.

Los medidores de combate soportados actualmente son Details,Recount, Skada & TinyDPS, uno de ellos tiene que estar activo.

Este complemento no require ni Elvui ni Tukui, pero si extan activos añadira un texto de datos a los mismos, usa la configuración en el juego de esos complementos para activarlo.

Incluye un texto de datos flotante que puedes colocar en cualquier lado de la pantalla.

Puedes pulsar botón derecho sobre el texto de datos para ver la ventana de configuración, los datos se graban en el sistema estandar de perfiles y pueden cambiar automáticamente al cambiar de espelización.

Nuevo: Incluye un modulo para salvar/anunciar récords de jefes. (Solo si tienes DBM o BigWigs)
