Cecile Meter Overlay : Léeme
Este complemento esta diseñado para mostrar la información de cualquier medidor de combate en un texto de datos. El concepto es que no necesitas tener abierto en tu pantalla tu medidor de combate.

Los medidores de combate soportados son Recount, Details!, Skada, TinyDPS y Numeration, uno de ellos tiene que estar activo.


ejemplo en pantalla

El texto de datos se puede personalizar mediante un sistema de etiquetas, desde tu DPS o el % de tu daño en la banda hasta la sanación total producida, el menú de configuración tiene opciones para personalizarlo o elegir uno de los formatos ya configurados.


ejemplo de texto de datos


opciones del texto de datos

Al pasar el ratón por encima del texto de datos una ventana flotante aparecerá donde puedes ver la información de tu banda o grupo. Puedes elegir que información mostrar desde el menú de configuración.


ejemplo de ventana flotante

El texto de datos se puede usar en Tukui, Elvui o en cualquier LDB broker como Chocolate Bar, pero no necesitas ninguno de estos complementos para usar Cecile_MeterOverlay.

Este complemento incluye un texto de datos flotante que puedes colocar en cualquier parte de la pantalla, también lo puedes personalizar desde el menú de configuración.


opciones del texto flotante

Incluye un modulo que salvar récords de jefes, pero solo si tienes DBM o BigWigs. Lo puedes configurar para auto anunciar nuevos récords en el chat del juego.


opciones de récords

Puedes explorar los records grabados con un ventana personalizada que te permitira anunciarlos por el chat en cualquier momento.


explorador de récords

Puedes pulsar botón derecho sobre el texto de datos para abrir la herramienta de configuración, alternativamente puedes usar los comandos: /cmo o /cecile_mo. La configuración se graba en el sistema de perfiles y puede cambiar por especialización.

NOTA:Este complemento no es para mostrar tu medidor de combate en el chat derecho de Elvui o Tukui, eso lo puedes hacer con AddOnSkin creado por Azilroka.
